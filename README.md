# 🦉 Owlie — JSOM AI Chatbot

AI chatbot frontend for the Naveen Jindal School of Management (JSOM) at UT Dallas. Clean dark-mode interface built in vanilla HTML/CSS/JS, deployed on Vercel.

## Overview

Owlie is a conversational AI assistant designed to help JSOM students and visitors get quick answers about courses, programs, faculty, and campus resources. The frontend provides a clean, responsive chat interface that connects to an AI backend.

---

## Features

- 💬 Real-time chat interface with streaming responses
- 🌙 Dark mode UI — easy on the eyes
- 📱 Responsive layout — works on desktop and mobile
- ⚡ Zero dependencies — pure HTML, CSS, and JavaScript
- 🚀 Deployed on Vercel (instant global CDN)

---

## Tech Stack

| Layer | Choice |
|---|---|
| Frontend | Vanilla HTML + CSS + JavaScript |
| Deployment | Vercel |
| Routing | `vercel.json` rewrites |

---

## Getting Started

```bash
git clone https://github.com/Prannay-tech/owl_frontend.git
cd owl_frontend

# Open directly in browser
open index.html

# Or serve locally
npx serve .
```

## Deployment

Deployed automatically via Vercel on push to main. Configuration in `vercel.json`.
